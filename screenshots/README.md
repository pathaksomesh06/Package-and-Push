# Screenshots

This directory contains screenshots of the Package & Push application.

## Required Screenshots

Please add the following screenshots to this directory:

1. **main-screen.png** - Welcome screen with Microsoft Entra ID sign-in button
2. **package-search.png** - Package search interface showing filtered results
3. **package-details.png** - Package detail view with build button
4. **app-configuration.png** - App configuration screen with metadata fields
5. **deployment-progress.png** - Deployment progress modal during upload
6. **version-update.png** - Version update confirmation dialog
7. **success-screen.png** - Success screen after successful deployment

## Screenshot Guidelines

- Use PNG format
- Recommended size: 1280x800 or higher
- Capture the full application window
- Ensure text is readable
